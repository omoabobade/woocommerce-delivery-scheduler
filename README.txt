== Description ==
A woo-commerce extension for scheduling product delivery. Customer can set an order as one-time order or a recurrent order.
On the checkout page customer have the choice to choose between recurrent delivery or one time delivery?
If auto delivery customer should select delivery frequency ie Weekly or Monthly
Customer should view Delivery schedule and choose the day and hour they would like the item delivered to them (Time Slot).
All the delivery information are accessible on order page in admin and also in customer own order page


== Installation ==

1. Upload `woocommerce-delivery-scheduler` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And you are good to go.
